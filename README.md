# E-commerce Backend Project

This is a side project e-commerce backend built with Node.js and TypeScript.

## Getting Started

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/thynamoral/e-commerce-backend.git
   cd e-commerce-backend
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Environment setup:
   ```
   cp .env.example .env.development
   ```
   Then edit the `.env.development` file with your specific configuration values.

### Running the Application

#### Development Mode
```
npm run dev
```

## Environment Variables

See `.env.example` for the required environment variables. The file includes:
- Database connection string
- JWT secret
- API keys for third-party services
- Server configuration
